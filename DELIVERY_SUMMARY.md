```bash

npm install
cp .env.example .env
# Edit .env with your Alchemy API key
```

### 2. Deploy Smart Contract
Go to [Remix IDE](https://remix.ethereum.org)
- Create `DigitalWill.sol`
- Copy code from `contracts/DigitalWill.sol`
- Compile with v0.8.20
- Deploy to Sepolia via MetaMask

### 3. Start Frontend

